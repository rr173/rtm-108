🌐 多语言镜像模块演示

一份主文档可挂多份不同语言的镜像，主文档更新时自动标出过期段落，翻译人员只需同步待处理段落。

📖 功能说明

✅ 核心特性

  • 镜像不是副本：跟着主文档版本一起演进
  • 自动过期检测：主文档一更新，立刻标出过期段落
  • 段落级同步：只能补待同步段落，不能整份覆盖
  • 左右对照工作台：左边原文，右边译文，过期高亮

🔄 协同机制

  • 版本状态追踪：查看每种语言跟到主文档哪个版本
  • 同步进度：显示还有多少段落没同步完
  • 已同步标记：全部段落处理完才能标记已同步
  • 实时刷新：多人同开时，他人提交立刻刷新待处理列表
